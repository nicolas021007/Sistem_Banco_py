# Sistem_Banco_py
Esse projeto  será desenvolvido como projeto de conhecimento e aperfeiçoamento na lingagem de programação em python.

Propósito gerar um sistema onde o cliente possar:

-Insirir nome;

-Inserir idade, caso for menor de 18 anos não poderá ter conta no banco,
se tiver 18 anos a mais passará tranquilamente;

-Inserir quantidade de saldo;

-Poderá também enviar dinheiro ao banco

-Poderá visualizar quanto saldo tem em sua conta.


No momento terá somente essas coisas no banco!
